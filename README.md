# plataforma-de-inscripciones
es una plataforma para que los estudiantes se puedan inscribir a los cursos virtuales de ingles y obtener su certificado

# publico objetivo 
para los estudiantes que no tienen tiempo para asitir a los cursos de forma presencial o por la distancia facilitando el ingreso y el movimiento sobre el dominio de la lengua principal (el ingles), mejorando sus
habilidades adqueriendo conocimientos previos 

# funcionalidades 
la plataforma como tal debe de garantizar el ingreso del usuario contraseña y datos del estudiante y hacer la 
inscripcion de el curso para poder ingresar al curso y realizar las actividades, y asi obtener un promedio, al final del curso se obtendra un certificado


# Descripción con mas solidez

La plataforma está diseñada para ser fácil de usar y segura, permitiendo a los estudiantes gestionar su acceso, inscribirse en cursos y visualizar el listado de inscritos. Al finalizar el curso, el estudiante podrá obtener un certificado

# tecnologias usadas

-html como estructura base de la pagina web

-css para darle estilo ala pagina

-javascript para darle funcionalidades a los botones y estilos
